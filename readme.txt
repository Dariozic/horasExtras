Before starting the server, you must go inside the folder: "PruebaNAM" and run: "npm i"

To set the server running make sure to run mongo service before 
Then, in a new terminal go inside PruebaNAM and run: "npm run dev" --To run the server.
After that in a new terminal inside the folder PruebaNAM/frontend run: "ng serve --open".

You can use the form to create a new register.
The right side of the page shows the registers you create.
Below the form, you will find the data retrieved from Json Place Holder.
